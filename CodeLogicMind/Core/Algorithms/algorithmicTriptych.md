# 📐🚕💨 Алгоритмический триптих

Метод разбора алгоритмов: три колонки — три уровня понимания.

## 📐 Шаблон

| Псевдокод (словами) | Код (Swift) | Схема (блок) |
|---------------------|-------------|--------------|
| Начало | `func name() {` | 🔵 Начало |
| Шаг 1 | `...` | ⬜ Действие |
| Условие | `if ...` | 💎 Ромб |
| Действие 2 | `...` | ⬜ Действие |
| Возврат | `return ...` | ⬜ Вернуть |
| Конец | `}` | 🔵 Конец |

---

## 📌 Пример: `sumOfUniqueOnce`

| Псевдокод | Код | Схема |
|-----------|-----|-------|
| Начало | `func sumOfUniqueOnce(_ nums: [Int]) -> Int {` | 🔵 Начало |
| Создать словарь | `var counts: [Int: Int] = [:]` | ⬜ Инициализация |
| Цикл по числам | `for num in nums {` | 🔁 Цикл |
| Обновить счётчик | `  counts[num, default: 0] += 1` | ⬜ Действие |
| Конец цикла | `}` | 🔁 Конец цикла |
| Фильтр частоты == 1 | `counts.filter { $0.value == 1 }` | 💎 Условие |
| Извлечь ключи | `      .map { $0.key }` | ⬜ Действие |
| Суммировать | `      .reduce(0, +)` | ⬜ Действие |
| Конец | `}` | 🔵 Конец |
---

## ✅ Использование

1. Берёшь функцию
2. Разбиваешь на логические шаги
3. Заполняешь три колонки:
   - **Словами** — что происходит
   - **Кодом** — как это выглядит в Swift
   - **Схемой** — какой блок (действие, условие, начало/конец)

💡 Три главных акцента для повторения:split(separator:) → всегда возвращает массив [Substring]. Для ключа словаря не забываем делать String(...).dict[key, default: []].append(...) → единственно верный и безопасный способ добавлять элементы в массив внутри словаря.
