# 🧠 Алгоритмический триптих

Метод разбора алгоритмов: три колонки — три уровня понимания.

## 📐 Шаблон

| Псевдокод (словами) | Код (Swift) | Схема (блок) |
|---------------------|-------------|--------------|
| Начало | `func name() {` | 🔵 Начало |
| Шаг 1 | `...` | ⬜ Действие |
| Условие | `if ...` | 💎 Ромб |
| Действие 2 | `...` | ⬜ Действие |
| Возврат | `return ...` | ⬜ Вернуть |
| Конец | `}` | 🔵 Конец |

---

## 📌 Пример: `sumOfUniqueOnce`

| Псевдокод | Код | Схема |
|-----------|-----|-------|
| Начало | `func sumOfUniqueOnce(_ nums: [Int]) -> Int {` | 🔵 Начало |
| Создать словарь для счётчиков | `var counts: [Int: Int] = [:]` | ⬜ Инициализация |
| Пройти по всем числам | `for num in nums {` | 🔁 Цикл |
| Увеличить счётчик числа | `counts[num, default: 0] += 1` | ⬜ Действие |
| Конец цикла | `}` | 🔁 Конец цикла |
| Оставить числа с частотой == 1 | `.filter { $0.value == 1 }` | 💎 Условие |
| Взять сами числа | `.map { $0.key }` | ⬜ Действие |
| Сложить их | `.reduce(0, +)` | ⬜ Действие |
| Вернуть результат | `return ...` | ⬜ Возврат |
| Конец | `}` | 🔵 Конец |

---

## ✅ Использование

1. Берёшь функцию
2. Разбиваешь на логические шаги
3. Заполняешь три колонки:
   - **Словами** — что происходит
   - **Кодом** — как это выглядит в Swift
   - **Схемой** — какой блок (действие, условие, начало/конец)
