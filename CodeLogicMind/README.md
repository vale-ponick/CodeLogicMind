**Уровень:** 🟢 Easy

---

## 1. ЧТО ВИДИТ ПОЛЬЗОВАТЕЛЬ
- [ ] Экран: одна цифра по центру + одна кнопка "нажать"
- [ ] Цифра начинается с 0
- [ ] Кнопка под цифрой (по центру)

---

## 2. ЧТО ДЕЛАЕТ ПРИЛОЖЕНИЕ
- [ ] При нажатии на кнопку цифра увеличивается на 1
- [ ] Максимального значения нет

---

## 3. ГРАНИЧНЫЕ СЛУЧАИ
- [ ] Цифра не уходит в минус (кнопки "-" пока нет)

---

## 4. ПЛАН ДЕКОМПОЗИЦИИ
- [ ] Переменные: `@State private var count = 0`
- [ ] UI: `Text(count)`, `Button("Нажать")`
- [ ] Логика: `count += 1`

---

## 5. ПСЕВДОКОД
На экране: цифра (0)

Под ней: кнопка "Нажать"

Нажали → цифра увеличилась на 1

Повторять бесконечно

•    Работа с @State
•    Базовый SwiftUI (Text, Button, VStack)
•    Обработка нажатий

•    Сделайте цифру анимированной при изменении (.animation())
•    Добавьте вторую кнопку Сброс, которая ставит 0
